Kiến Trúc Tự Động Hóa Thực Tế (Production Blueprint)

Dưới đây là sơ đồ kỹ thuật chi tiết giúp bạn triển khai quy trình này trên hệ thống của công ty bạn bằng cách tích hợp **Google Workspace (Gmail)** hoặc **Microsoft 365 (Outlook)** với **LLM (như Gemini API)**.

Sơ đồ luồng dữ liệu tự động

1. Người dùng tải File

Tải CCCD, Offer lên Server nội bộ

2. AI Parser (Gemini)

Đọc text/ảnh trích xuất JSON data

3. Email Service API

Tự động gửi mail đính kèm cho HRD

4. Webhook / Mail Listener

Lắng nghe email trả lời tự động

5. Hoàn thành luồng

Ký số và lưu trữ hợp đồng

Chi tiết các bước cài đặt

  1. Tự động đọc và trích xuất dữ liệu:

    Sử dụng API Gemini 1.5 Flash / Document AI. Bạn chỉ cần gửi ảnh CCCD, PDF offer cho API với Prompt: "Hãy trích xuất các thông tin sau dạng JSON: Họ tên, Ngày sinh, CCCD, Mức lương...". Trực quan hóa và cho phép kiểm tra thủ công (Human-in-the-loop) để đảm bảo chính xác 100%.

  2. Gửi Email tự động (Mailing System):

    Sử dụng thư viện hoặc API như Nodemailer (Node.js), SendGrid hoặc chính Microsoft Graph / Gmail API để gửi thư trực tiếp từ tài khoản của bạn. Đính kèm hợp đồng dạng DOCX/PDF tự động sinh ra từ mã nguồn.

  3. Hộp thư chung & Lắng nghe email phản hồi (Webhook/Mail Polling):

    Đăng ký một Webhook lắng nghe email đến (ví dụ Microsoft Graph Webhook hoặc Gmail Pub/Sub). Khi có email trả lời từ vananh@company.com hoặc CEO, Webhook sẽ đẩy nội dung email về API backend của bạn.

  4. Bộ máy Phân tích ý định Email (Gemini Intent Analysis):

    Thay vì so khớp từ khóa cứng nhắc, trong thực tế bạn có thể truyền nội dung email đó cho Gemini API với yêu cầu phân loại:

    Phân tích email phản hồi sau và trả về định dạng JSON:
Nội dung: "[Nội dung Email của HRD]"
Yêu cầu trả về:
{
  "decision": "approved" | "rejected",
  "reason": "Lý do từ chối (nếu có, để trống nếu duyệt)"
}
  5. Quản lý trạng thái & Cập nhật hợp đồng:

    Tương ứng với JSON nhận được từ AI, server sẽ đổi trạng thái hợp đồng trong cơ sở dữ liệu. Nếu từ chối, gửi cảnh báo Teams/Slack/Email ngược lại cho bạn. Nếu đồng ý, tự động kích hoạt bước tiếp theo (gửi email cho CEO).
